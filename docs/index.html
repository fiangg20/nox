Nox

Nox is a premium UI framework for Roblox that fully adapts Material Design 3 (M3) guidelines. It comes equipped with Ripple animations, a dynamic color system, auto-saving configurations, and a highly responsive built-in search feature.

rocket_launch Initialization

Load the library into your script. This library will automatically download Google Sans and Material Icons assets into your workspace folder if they don't already exist.

local Nox = loadstring(game:HttpGet("https://raw.githubusercontent.com/fiangg20/nox/refs/heads/main/source.lua"))()

web_asset Window Base

Creates the main Window/GUI. This is the foundation of your script. It includes a built-in Search Bar that automatically registers all elements.

ArgumentTypeDescription
TitlestringThe main title of the application.
IconstringThe main icon of the application.
SizeX, SizeYnumberInitial window size (default 380x520).
ToggleKeyEnum.KeyCodeKeyboard key to Minimize/Maximize the UI (Default: K).
ThemestringInitial color theme. Options: Purple, Blue, Red, Green, Orange, Default.
SearchbooleanDisplays a Search Bar at the top of the UI (Default: false).
UnlockMousebooleanUnlock mouse while in FPS (Default: true).
local Window = Nox:Create({
    Title = "Nox Hub",
    SizeX = 380,
    SizeY = 520,
    ToggleKey = Enum.KeyCode.K,
    Theme = "Purple", 
    Search = true,
    UnlockMouse = true,
    SearchPlaceholder = "Search features...",
    OnSearch = function(text)
    end,
    OnClose = function()
        print("UI Closed!")
    end
})
Window:SetTitle("Nox Hub V2")

save Config System System

Nox features a built-in JSON configuration management system. Any UI element initialized with a Flag = "FlagName" parameter will have its value automatically saved/loaded to this system.

Window:SaveConfig("MyConfig")
Window:LoadConfig("MyConfig")
local savedConfigs = Window:GetConfigs() 
for i, name in ipairs(savedConfigs) do
    print(i, name)
end

palette Theme System System

You can change the color palette in real-time with smooth transitions, or create your own custom themes using Material 3 Color Roles (Background, Text, Primary, etc).

Window:ChangePalette("Red") 
Window:AddTheme("MyCustomDark", {
    Background = Color3.fromRGB(20, 20, 20),      
    Text = Color3.fromRGB(240, 240, 240),         
    Primary = Color3.fromRGB(0, 170, 255),        
    TextOnPrimary = Color3.fromRGB(10, 10, 10),   
    Surface = Color3.fromRGB(35, 35, 35),         
    Outline = Color3.fromRGB(120, 120, 120)       
})
Window:ChangePalette("MyCustomDark")

tab Tabs Layout

Divides content into multiple pages (Tabs). When AddTab is called for the first time, the UI will automatically spawn a tab navigation area below the header.

local MainTab = Window:AddTab({
    Title = "Main",
    Icon = "home" 
})
local SettingsTab = Window:AddTab({
    Title = "Settings",
    Icon = "settings"
})
MainTab:SetText("Home")
Window:SelectTab("Settings")

emoji_symbols Icons System System

Nox fully integrates Google Material Icons. Any element that accepts an Icon parameter can use official Material Symbol names directly without needing an image ID. It also supports custom Roblox Image IDs!

    Text = "Settings",
    Icon = "settings", 
    Callback = function() end
})

    Text = "Custom Logo",
    Icon = "rbxassetid://1234567890", 
    Callback = function() end
})

smart_button Button Element

Interactive buttons with M3 Ripple effects. Supports 4 visual types.

Type (String)Appearance Description
"filled"Primary color background, TextOnPrimary text color. (Default)
"tonal"Transparent Surface color background, primary text color.
"outlined"No background, features a border outline.
"text"Text (and icon) only, no border/background, Primary color.
local Btn = MainTab:AddButton({
    Text = "Print Hello",
    Type = "filled", 
    Icon = "terminal", 
    Width = nil, 
    Callback = function()
        print("Button Clicked!")
    end
})
Btn:SetText("New Button Name")

toggle_on Switch Element

Toggle elements used to adjust Boolean values (True/False). Features the signature Material 3 capsule animation.

local AutoFarmToggle = MainTab:AddSwitch({
    Title = "Auto Farm",
    Default = false,
    ShowCheckIcon = true,
    Icon = "bolt",
    Flag = "Toggle_AutoFarm", 
    Callback = function(state)
        print("Auto Farm is now:", state)
    end
})
AutoFarmToggle:SetValue(true) 
AutoFarmToggle:SetText("Auto Farm [UPDATED]")

linear_scale Slider Element

Select number/float values. Uniquely, Nox supports various slider sizes ranging from XS to XL.

Size (String)Handle HeightContains Icon?
"xs"Small (16px)Yes
"s"Medium (24px)Yes
"m"Normal (40px)Yes (24px)
"l"Large (56px)Yes (24px)
"xl"Extra Large (96px)Yes (32px)
local SpeedSlider = MainTab:AddSlider({
    Title = "WalkSpeed",
    Min = 16,
    Max = 200,
    Default = 16,
    ShowValue = true,
    Size = "m", 
    Icon = "speed", 
    Flag = "Slider_WS",
    Callback = function(value)
        game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = value
    end
})
SpeedSlider:SetValue(50)
SpeedSlider:SetText("Run Speed")
SpeedSlider:SetIcon("directions_run")

text_fields Text Box Element

Text input field. Features the M3 Floating Label animations seamlessly.

local TargetBox = MainTab:AddTextBox({
    Title = "Target Player",
    SupportText = "Enter the name of the player you want to target", 
    Icon = "person",
    Flag = "Text_Target",
    Callback = function(text, enterPressed)
        print("Target set to:", text, "| Enter Pressed:", enterPressed)
    end
})
TargetBox:SetValue("RobloxPlayer123")
TargetBox:SetText("Target (Username)")

arrow_drop_down_circle Dropdown Element

Select one item from a drop-down list.

local LocationDrop = MainTab:AddDropdown({
    Title = "Teleport Location",
    Options = {"Spawn", "Shop", "Arena", "Boss Area"},
    Default = 1, 
    Icon = "map",
    Flag = "Drop_Location",
    Callback = function(selectedOption)
        print("Teleporting to:", selectedOption)
    end
})
LocationDrop:SetValue("Arena") 
LocationDrop:Refresh({"New Area 1", "New Area 2"}, 1) 
LocationDrop:SetText("Select Location")

radio_button_checked Radio Group Element

Similar to a dropdown (selecting one from many options), but displayed directly on the screen.

    Default = 1,
    Flag = "Radio_Hitbox",
    Callback = function(selectedString, index)
        print("Hitbox set to:", selectedString)
    end

local currentString, currentIndex = ModeRadio:GetValue()
ModeRadio:SetValue(2)

    Title = "Enable Noclip",
    Default = false,
    Flag = "Check_Noclip",
    Callback = function(state)
        print("Noclip:", state)
    end

local isChecked = NoclipCheck:GetValue()
NoclipCheck:SetValue(true)
NoclipCheck:SetText("Noclip (Walk through walls)")

smart_button Chip Group Element

Groups of small buttons (Chips) that can be horizontally scrolled. Supports Single-Select or Multi-Select modes.

local FilterChips = MainTab:AddChipGroup({
    Title = "Item Rarity Filter",
    Type = "outlined", 
    Multiselect = true, 
    Options = {
        "Common", 
        "Uncommon", 
        {Text = "Rare", Icon = "star"}, 
        {Text = "Legendary", Icon = "diamond"}
    },
    Flag = "Chips_Filter",
    Callback = function(selectedItems)
        for _, v in ipairs(selectedItems) do print(v) end
    end
})
local selected = FilterChips:GetSelected()
FilterChips:SetValue({
    [1] = true,  
    [3] = false  
})

notifications Notification (Snackbar) Overlay

Displays a notification Snackbar at the bottom of the screen. Supports adding interactive action buttons.

Window:Notify({
    Text = "Successfully loaded auto configuration.",
    Duration = 5, 
    Actions = {
        {
            Text = "Undo",
            Callback = function()
                print("Action Cancelled!")
            end
        }
    }
})

chat_bubble Dialog Overlay

Pops up a modal dialog in the center of the screen, blocking interactions behind it until a button is pressed.

Window:AddDialog({
    Title = "Ban Warning",
    Description = "This feature is highly risky and may lead to account bans. Proceed?",
    Buttons = {
        {
            Text = "Cancel", 
            Type = "text", 
            Callback = function() print("Cancelled") end
        },
        {
            Text = "I Understand, Proceed", 
            Type = "filled", 
            Callback = function() print("Executing risky feature...") end
        }
    }
})
