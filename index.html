Nox

A premium UI framework for Roblox that fully adapts Material Design 3 (M3) guidelines. It comes equipped with Ripple animations, a dynamic color system, auto-saving configurations, and a highly responsive built-in search feature.

rocket_launch Initialization

Load the library into your script. This library will automatically download Google Sans and Material Icons assets into your workspace folder if they don't already exist.

local Nox = loadstring(game:HttpGet("https://raw.githubusercontent.com/fiangg20/nox/refs/heads/main/source.lua"))()

web_asset Window Base

Creates the main Window/GUI. This is the foundation of your script. It includes a built-in search bar that automatically registers all elements.

ArgumentTypeDescription
TitlestringThe main title of the application.
SizeX, SizeYnumberInitial window size (default 380x520).
ToggleKeyEnum.KeyCodeKeyboard key to Minimize/Maximize the UI (Default: K).
ThemestringInitial color theme. Options: Purple, Blue, Red, Green, Orange, Default.
SearchbooleanDisplays a Search Bar at the top of the UI (Default: false).
local Window = Nox:Create({
    Title = "Nox Hub",
    SizeX = 380,
    SizeY = 520,
    ToggleKey = Enum.KeyCode.RightControl,
    Theme = "Purple", 
    Search = true,
    SearchPlaceholder = "Search features...",
    OnSearch = function(text)
        
    end,
    OnClose = function()
        print("UI Closed!")
    end
})
Window:SetTitle("Nox Hub V2")

save Config System System

Nox features a built-in JSON configuration management system. Any UI element initialized with a Flag = "FlagName" parameter will have its value automatically saved/loaded to this system.

Window:SaveConfig("MyConfig")
Window:LoadConfig("MyConfig")
local savedConfigs = Window:GetConfigs() 
for i, name in ipairs(savedConfigs) do
    print(i, name)
end

palette Theme System System

You can change the color palette in real-time with smooth transitions, or create your own custom themes using Material 3 Color Roles (Background, Text, Primary, etc).

Window:ChangePalette("Red") 
Window:AddTheme("MyCustomDark", {
    Background = Color3.fromRGB(20, 20, 20),      
    Text = Color3.fromRGB(240, 240, 240),         
    Primary = Color3.fromRGB(0, 170, 255),        
    TextOnPrimary = Color3.fromRGB(10, 10, 10),   
    Surface = Color3.fromRGB(35, 35, 35),         
    Outline = Color3.fromRGB(120, 120, 120)       
})
Window:ChangePalette("MyCustomDark")

tab Tabs Layout

Divides content into multiple pages (Tabs). When AddTab is called for the first time, the UI will automatically spawn a tab navigation area below the header.

local MainTab = Window:AddTab({
    Title = "Main",
    Icon = "home" 
})
local SettingsTab = Window:AddTab({
    Title = "Settings",
    Icon = "settings"
})
MainTab:SetText("Home")
Window:SelectTab("Settings")

emoji_symbols Icons System System

Nox fully integrates Google Material Icons. Any element that accepts an Icon parameter can use official Material Symbol names directly without needing an image ID. It also supports custom Roblox Image IDs!
